## 1. 产品概述

一款在线证件照生成工具，用户上传手机拍摄的纯色背景人物大头照片，选择国家/规格和背景颜色，一键生成符合标准的证件照文件并导出使用。

### 核心价值

- 无需安装软件，浏览器即可使用
- 支持多国证件照规格
- 本地/服务端双模式处理，平衡隐私与效果
- PWA 支持移动设备访问

---

## 2. 用户流程

```
打开网站 → 上传照片 → 选择国家 → 选择规格 → 选择背景色 → 选择/自动选择处理模式 → 一键生成 → 预览 → 下载
```

## 4. 系统架构

### 4.1 整体架构

```
Client (Browser: React SPA + PWA)
    │
    ├── 本地模式: MediaPipe Selfie Segmentation (WASM) → Canvas 合成
    │
    └── 服务端模式: HTTP REST API → Python FastAPI
                            │
                     ┌──────┴──────┐
                     │  rembg      │
                     │  Pillow     │
                     │  OpenCV     │
                     └──────┬──────┘
                            │
                     临时文件 (处理完即删)
```

### 4.2 前后端分工

| 处理阶段 | 本地模式 (MediaPipe) | 服务端模式 (rembg) |
|---------|-------------------|-------------------|
| 人像分割 | 浏览器 WASM 推理 | Python ONNX 推理 |
| 背景替换 | Canvas 像素操作 | Pillow + 羽化处理 |
| 尺寸裁剪 | Canvas 模板裁剪 | Pillow 精确裁剪 |
| 排版合成 | Canvas 多格排版 | Pillow 排版 |
| 导出 | Blob → 下载 | 流式返回 |
| 照片存储 | 无（全内存） | 临时文件，请求结束即删 |

### 4.3 自动模式判断逻辑

```
上传照片 → 质量检测
  ├─ 清晰度好 + 背景简单 → 本地模式（快速）
  ├─ 清晰度差 + 背景复杂 → 服务端模式（高精度）
  └─ 不确定 → 默认本地，提示用户可切换
```

## 6. 数据处理流水线

```
上传照片
  │
  ▼
1. 预处理
   ├─ 读取 EXIF，自动旋转修正
   ├─ 压缩至最长边 ≤ 2000px
   └─ 人脸检测 + 智能裁剪建议
  │
  ▼ (自动/手动选择模式)
  ├─ 本地 → MediaPipe 分割 (WASM)
  └─ 服务端 → rembg 分割 (ONNX)
  │
  ▼
2. 背景替换
   ├─ 生成 alpha mask
   ├─ 边缘羽化
   ├─ 按色值渲染纯色背景
   └─ 头发丝级别优化（服务端模式）
  │
  ▼
3. 裁剪 + 构图
   ├─ 按规格计算像素尺寸
   ├─ 基于人脸检测调整头部占比
   └─ 精确裁剪到目标尺寸 (300dpi)
  │
  ▼
4. 排版输出 (可选)
   ├─ 按 6寸相纸 (102×152mm) 布局
   ├─ 自动排列多张
   └─ 导出 JPG/PNG
  │
  ▼
预览 → 下载
```

## 9. 非功能需求

- 首屏加载 < 3s（优化 WASM 模型懒加载）
- 本地处理：从上传到出图 < 5s（普通照片）
- 服务端处理：API 响应时间 < 10s
- 支持主流浏览器：Chrome、Edge、Safari、Firefox
- 移动端适配：响应式布局 + PWA
- 隐私：不保存任何用户照片（服务端处理完即删）
